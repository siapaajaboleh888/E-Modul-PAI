# E-Modul Interaktif PAI

Aplikasi web untuk E-Modul Interaktif Pendidikan Agama Islam (PAI) dengan integrasi Problem-Based Learning (PBL) dan nilai keberagamaan transformatif di PTU.

## Stack Teknologi

- Node.js + Express (back-end)
- EJS (server-side rendering)
- Bootstrap (UI responsive)
- SQLite (database file-based)

## Cara Menjalankan

1. Pastikan Node.js sudah terpasang di komputer.
2. Buka terminal di folder `d:\EModul`.
3. Install dependensi:
   ```bash
   npm install
   ```
4. Jalankan aplikasi (development):
   ```bash
   npm run dev
   ```
5. Buka browser dan akses: `http://localhost:3000`.

## Fitur Utama (Rancangan)

- Autentikasi (Mahasiswa, Dosen, Admin)
- Dashboard Mahasiswa (modul terakhir, progres, aktivitas, PBL)
- Dashboard Dosen/Admin (statistik, manajemen modul, materi, PBL, aktivitas, nilai, laporan)
- Modul pembelajaran, skenario PBL, aktivitas interaktif, progres belajar, glosarium, manajemen nilai keberagamaan.
